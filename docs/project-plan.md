# Momento Project Plan

## Phase 1: Project Preparation

- Create the GitHub repository
- Clone the repository locally
- Prepare README.md
- Prepare .gitignore
- Create docs folder
- Add the general project plan

## Phase 2: Technical Initialization

- Create frontend with React, TypeScript, and Tailwind CSS
- Create backend with Node.js, Express, and TypeScript
- Configure PostgreSQL and Prisma
- Create the first test API route

## Phase 3: Database Modeling

- Create Prisma models
- Add enums
- Run the first database migration

## Phase 4: Authentication

- User registration
- User login
- Password hashing with bcrypt
- JWT authentication
- Role-based access control

## Phase 5: Social Features

- Create posts
- Upload multiple post images
- Display posts
- Add comments
- Add likes/reactions
- Follow and unfollow users
- Report posts and users

## Phase 6: Provider Features

- Submit provider request
- Add CIN number and CIN picture
- Admin review of provider requests
- Create provider profile after approval
- Manage services and service images

## Phase 7: Admin Features

- Manage users
- Block or unblock accounts
- Manage provider requests
- Manage reports
- Moderate content

## Phase 8: SuperAdmin Features

- View global statistics
- Monitor platform activity
- View logs
- Manage important system actions

## Phase 9: Notifications

- Notification for likes
- Notification for comments
- Notification for provider request status
- Admin notifications

## Phase 10: Chat and Messaging

- Create chat-service
- Add conversation and message models
- Create conversation endpoints
- Create send message endpoint
- Create messages list endpoint
- Create chat frontend page
- Connect Contact Provider button to chat
- Add real-time chat later as a future improvement

## Phase 11: Testing

- Manual testing
- Backend route testing
- Frontend component testing
- Optional end-to-end testing

## Phase 12: Deployment

- Prepare environment variables
- Create production PostgreSQL database
- Deploy backend
- Deploy frontend
- Configure image hosting
- Test the online version

## Phase 13: Documentation

- Final README
- Screenshots
- Installation guide
- Deployment guide
- Final report
- Presentation
